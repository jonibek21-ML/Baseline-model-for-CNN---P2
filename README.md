# Baseline-model-for-CNN---P2

🧠 Skin Disease Classification (CNN | PyTorch)
📌 Loyiha haqida

Ushbu loyiha Kaggle Skin Disease Classification dataset asosida teri kasalliklarini rasmlar orqali CNN (Convolutional Neural Network) yordamida klassifikatsiya qilish uchun ishlab chiqilgan.
